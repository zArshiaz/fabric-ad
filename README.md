# Fabric Admin Dashboard

---

> This project is an admin dashboard for the [Fabric Store](https://github.com/zArshiaz/fabric-store) project.
> This dashboard allows administrators to manage store data through a dedicated interface.

⚠️ **Important:** This project depends on the Fabric Store backend repository. Make sure the backend is cloned and running before starting the admin dashboard.

Backend repository:
https://github.com/zArshiaz/fabric-store

---

![project image](./public/images/img.png)

---

## Requirements

* Node.js 18+ (recommended)
* Angular CLI
* Fabric Store backend running locally on port 4000

Install Angular CLI if you don't have it:

```bash
npm install -g @angular/cli
```

---

## Setup Instructions

### 1. Clone and run the backend (Fabric Store)

```bash
git clone https://github.com/zArshiaz/fabric-store.git
cd fabric-store/back
npm install
npm start
```

Make sure the backend is running before starting the admin dashboard.

---

### 2. Run the admin dashboard

Install dependencies:

```bash
npm install
```

Start the Angular development server:

```bash
ng serve
```

---

## Access the Application

Open your browser and navigate to:

```
http://localhost:4200
```
